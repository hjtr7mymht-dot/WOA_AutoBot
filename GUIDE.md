# 📖 WOA AutoBot 使用说明

> 你可以自由编辑此文件（`GUIDE.md`）来更新使用说明内容。

---

## 🚀 快速开始

1. **启动模拟器**：打开 MuMu 模拟器 12，运行游戏并进入主界面
2. **启动脚本**：运行 `WOA_AutoBot.exe`，点击 **刷新设备** 选择模拟器
3. **点击启动**：点击 **▶ 启动** 按钮，脚本开始自动处理任务

---

## ⚙️ 主要功能

| 功能 | 说明 |
|------|------|
| 进场处理 | 自动分配停机位、确认降落 |
| 离场处理 | 自动推出、滑行、起飞，支持领奖流程 |
| 地勤分配 | 自动滑动滑块分配地勤人员 |
| 除冰/维修 | 自动处理除冰和维修任务 |
| 塔台延时 | 自动检测红灯并全部激活控制器 |
| 车辆购买 | 车辆不足时自动购买 |

---

## 🎛️ 设置说明

### 筛选模式
- **仅待处理**：只显示待处理任务（默认）
- **仅停机位**：只显示停机位任务（塔台全开时生效）

### 不起飞模式
- 开启后只处理进港和停机位任务，不处理起飞
- 自动在"待降落"和"停机坪"之间轮切

### 防卡死
- 连续超时自动触发紧急恢复
- 可在设置中调整触发阈值（3-20）

### 自动延时塔台
- 设置延时次数后自动检测并激活全部控制器
- 红灯最高优先级，检测到立即处理

## ❓ 常见问题

**Q: 截图失败怎么办？**
A: 尝试切换截图方案：设置 → 截图方案 → 选择 ADB / nemu_ipc / DroidCast

**Q: 触控不生效？**
A: 尝试切换触控方案：设置 → 触控方案 → 选择 ADB / uiautomator2

**Q: 脚本频繁关闭窗口？**
A: 检查是否在正确的游戏界面（机场主界面），确保任务列表可见
